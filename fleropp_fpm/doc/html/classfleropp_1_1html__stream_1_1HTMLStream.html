Detailed Description

template<fleropp::util::StringLiteral StartT, fleropp::util::StringLiteral EndT, typename EndTagAction = void>
class fleropp::html_stream::HTMLStream< StartT, EndT, EndTagAction >

Class template that represents an output stream designed with HTML divs in mind.

Template Parameters
StartTString representation of the start tag (e.g., "<html>").
EndTString representation of the end tag (e.g., "</html>").
EndTagAction(optional) Tag type indicating behavior on receipt of EndTag from gen_end_tag (default = void).

Friend stream insertion operator that recieves end tag and conditionally writes to output stream.

Friend stream insertion operator that recieves end tag. The behavior of this function depends on std::is_same<EndTagAction, X>::value, for some tag type X. For example, std::is_same_v<EndTagAction, DumpOnEnd>, the stream will be written to the global output stream upon receipt of the end tag. By default, nothing is done upon receipt of the end tag.
