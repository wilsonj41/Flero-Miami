Closes the library and frees its resources.

This function unloads the library opened by open_lib() and frees all resources allocated by it. No effect will be had if the library was not already opened. Errors will be sent to standard error.

Implements fleropp::fpm::ISOLoader< T >.

Function that loads the shared library into memory for symbol access.

Loads shared library into memory so that symbols can be accessed. Will not open if already opened and will output error if any unresolved symbols are found.
