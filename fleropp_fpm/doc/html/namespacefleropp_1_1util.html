consteval std::size_t fleropp::util::count_overridden ( const member_map_t< Base, Derived, N, Args... > &  mmap)

Count the number of overridden member functions at compile time.

Given a compile-time map of pointers-to-member functions to their string representations for a CRTP class, this function counts the number of them that have been overridden.

The parameters for each of the member functions must be the same.

NThe length of the map of member functions.

Returns

consteval std::array< std::string_view, M > fleropp::util::get_overridden ( const member_map_t< Base, Derived, N, Args... > &  mmap)

List the overridden member functions at compile time.

Given a compile-time map of pointers-to-member functions to their string representations for a CRTP class, this function provides a std::array<std::string_view> containing the string representations of only those that have been overridden.

The parameters for each of the member functions must be the same.

Template Parameters
BaseThe type of the base class
DerivedThe type of the derived class
NThe length of the map of member functions.
MThe number of overridden member functions.
ArgsThe arguments to each of the member functions.
Parameters
[in]mmapThe map of member function pointers to their string representations.
Returns
An array containing the names of only the overridden member functions.
